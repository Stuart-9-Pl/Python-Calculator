# Simple Calculator

A minimal Python project for basic arithmetic operations.

## Functions
- `add(a, b)` – adds two numbers
- `subtract(a, b)` – subtracts b from a
- `multiply(a, b)` – multiplies two numbers
- `divide(a, b)` – divides a by b

## How to run
```bash
python calculator.py
